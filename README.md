#仿西游黄金矿工
- 该游戏模仿了九秒社团上的开源项目西游黄金矿工   
- 游戏中所有的美术和音效均来自九秒社团的开源项目西游黄金矿工   
- 九秒社团的游戏使用的是Cocos2d-x-2.1.5开发的，而我这个游戏UI部分使用CocosStudio搭建的，游戏逻辑部分使用Cocos2d-js-3.6开发的   
- 所以这款游戏可以同时移植到web平台和native平台上
